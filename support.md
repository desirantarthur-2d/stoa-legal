---

## Questions fréquentes

**Comment supprimer mon compte ?**
Depuis l'application : Profil → Réinitialiser profil. Cette action supprime définitivement votre compte et toutes vos données associées.

**Comment annuler mon abonnement Premium ?**
Les abonnements sont gérés directement par Apple. Rendez-vous dans les Réglages de votre iPhone → votre nom → Abonnements, puis sélectionnez Stoa pour gérer ou annuler votre abonnement.

**J'ai un problème technique avec l'app, que faire ?**
Contactez-nous à l'adresse ci-dessus en décrivant le problème rencontré (idéalement avec une capture d'écran si possible). Nous vous répondrons rapidement.

**Comment fonctionne le mentor vocal ?**
Le mentor génère chaque jour une analyse personnalisée de la citation du jour, basée sur votre profil. Cette analyse peut être écoutée à voix haute directement dans l'application.
